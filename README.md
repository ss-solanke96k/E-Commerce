
# 🛒 E-Commerce Website

A simple and responsive **E-Commerce Website** built using **HTML, CSS, and JavaScript**.
This project includes core shopping features like **Add to Cart** and **Remove from Cart** functionality.

---

## 🚀 Features

* 🛍️ Product Listing Page
* ➕ Add to Cart functionality
* ➖ Remove from Cart functionality
* 🧮 Dynamic Cart Item Count
* 📱 Responsive Design
* 🎨 Modern UI with Hover & Active Effects

---

## 🛠️ Technologies Used

* **HTML5** – Structure
* **CSS3** – Styling & Responsive Design
* **JavaScript (Vanilla JS)** – Cart Logic & DOM Manipulation

---

## 📂 Project Structure

```
E-Commerce-Website/
│
├── index.html
├── style.css
├── script.js
```

---

## ⚙️ How It Works

* Products are displayed dynamically.
* When user clicks **Add to Cart**, the item is stored in a cart array.
* Clicking **Remove** deletes the product from cart.
* Total price updates in real-time.
